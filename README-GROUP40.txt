 Instructions to Run the Project:

1.After unzipping, In the `backend` folder, open the `config` directory, then locate `db.js`. Inside `db.js`, replace the MongoDB connection URL with your own MongoDB Compass URL, including the project name.  
   Example:  
   
   await mongoose.connect('mongodb://localhost:27017/final-project');
  
   
2. Open the `backend` folder in its integrated terminal and run the following commands:  

   npm install(for installing node modules)
   node server.js
   
   
3. Open a new terminal, navigate to the `frontend` folder, and run the following commands to start the frontend server:  
  npm install(for installing node modules)
   
   npm start
   
   This will launch the project on `localhost`.

4. If you already have an account, log in using the appropriate email and password. If not, sign up using the provided forms. Once signed in, you will be redirected to the respective dashboard based on your role (Admin, Restaurant, or User).


About website:-**Project Description: Yummy Food Ordering Website**  

The Yummy Food Ordering Website is a robust platform that connects users, restaurants, and administrators for seamless online food delivery. Users can register, log in, and explore a wide variety of cuisines from nearby restaurants. The platform features a user-friendly **User Dashboard** where customers can browse menus, add items to the cart and place orders
Restaurants manage their listings, menus, and orders through a dedicated **Restaurant Dashboard**, allowing them to update food availability and prices dynamically. The **Admin Panel** offers complete control to the website administrators, enabling them to oversee user activities, approve or reject restaurant profiles, and monitor system performance.
